Fire Weather Index (FWI) Prediction System

Predicting the Fire Weather Index based on meteorological data

Welcome to the FWI Prediction Project

This project aims to predict the Fire Weather Index (FWI) using machine learning techniques. By inputting key meteorological factors such as temperature, relative humidity, wind speed, and rain, our model predicts the fire weather index which indicates fire risk.
